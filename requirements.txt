# at the license-selection gate: every Open3D wheel statically links Eigen,

# uses, LGPL unless built with -DEIGEN_MPL2_ONLY) — both explicitly
# disqualifying under this package's copyleft-anywhere-in-the-tree rule.
# The Point Cloud Library (PCL) has the same Eigen dependency and is
# excluded for the same reason. Instead this package composes small,
# individually-permissive, Eigen-free libraries — see below.
